# MPP Rocket League - Monorepo

Un monorepo contenant l'application client et serveur pour MPP Rocket League.

## Structure

- `client/` - Application Vue.js frontend
- `server/` - API Node.js/Express backend

## Prérequis

- Node.js >= 22.0.0
- Yarn >= 3.0.0

## Installation

```bash
yarn
```

## Scripts disponibles

### Développement

```bash
yarn dev

```

### Tests et qualité

```bash
# Linter tout le code
yarn lint

# Formatter tout le code
yarn format
```
